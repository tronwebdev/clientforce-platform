## BusinessContext field registry (DEC-024, owner-approved 2026-07-03)

Semantics: **Required** = must be filled **with citations** from READY sources
(workspace or agent layer); uncited → gap row ("Type it" / "✦ Let AI"), step-6
launch gated. **Recommended** = distilled + cited when evidence exists; never
gaps, never blocks. Field keys are stable identifiers shared by the distiller,
gap checker, planner, and Brand kit.

**Core (required for every goal):** `offer` · `usp` · `tone`.

**Workspace-level required for any email goal:** `company_address` — the postal
address CAN-SPAM requires in every unsubscribe footer. Asked **once at
workspace level**, never per agent; usually auto-distilled + cited from the
website's contact/footer pages, so it only gaps when truly absent. **P1.5's
footer consumes this field — never a placeholder.**
